# HRMIS

- 基于CUI交互
- 测试前准备：
  - 当前项目根目录存放员工初始信息`records.txt`
  - 编码格式均为UTF-8
- 完全功能测试，运行`/HRMIS-Se7en/src/edu/fzu/hrmis/SysLoader.java`即可
